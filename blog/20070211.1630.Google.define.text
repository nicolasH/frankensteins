Google define
Google Define is a search plugin for firefox. It is a very small modification of the default google firefox search plugin. The arguments will be passed to google with the "define:" prefix .   
It is composed of the <a href="http://nikolas13.chez-alice.fr/googleDefine/googledefine.xml">xml file</a> (use "save as", do not try to see it in a browser: it start with a #) and the <a href="http://nikolas13.chez-alice.fr/googleDefine/googledefine.png">icon</a> (<a onblur="try {parent.deselectBloggerImageGracefully();} catch(e) {}" href="http://nikolas13.chez-alice.fr/googleDefine/googledefine.png"><img style="cursor: pointer; width: 16px;" src="http://nikolas13.chez-alice.fr/googleDefine/googledefine.png" alt="" border="0" /></a>).  you have to place them in the "searchplugin" directory of the firefox installation.
