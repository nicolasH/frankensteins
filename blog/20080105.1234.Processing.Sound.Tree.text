*  Install [processing](http://www.processing.org)
*  Download the [3d sound tree source][3dSoundTree] (scroll down.)
*  Download the following libraries : 
   *  [minim](http://code.compartmental.net/tools/minim "sound manipulation library for processing")
   *  [controlP5](http://www.sojamo.de/libraries/controlP5/)
   *  [Tuple](http://screamyguy.net/Tuple/index.htm)
   *  [SGCamera](http://screamyguy.net/SGCamera/index.htm)
   *  [SGGui](http://screamyguy.net/SGGUI/index.htm)

note: all the screamyGuy stuff have to be unzipped from the command line on mac os x, otherwise the udecompress utility creates a directory with backslash in the name.
 
place all the directories containing the library directory in the processing library directory.

[3dSoundTree] : http://processing.org/discourse/yabb_beta/YaBB.cgi?board=Sound;action=display;num=1196771719 "3d tree growing from the sound"
